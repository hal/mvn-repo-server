HAL Build Proxy

Welcome, I am the HAL build proxy. I serve HAL web console builds and I don't wont to be pretty.

API Endpoints

URL Description Example
/latest Displays the version of the latest HAL release /latest
/release/:version Serves a specific HAL version (only works with version > 2.6.x) /release/2.6.5.Final
/snapshot/:version Serves a specific HAL snapshot (only works with version > 2.6.x) /snapshot/2.7.0-SNAPSHOT
